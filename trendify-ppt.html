Project Overview

What is
Trendify?

A modern, Gen-Z focused fashion e-commerce platform combining slick UI design with AI-powered shopping tools — built as a complete production-ready MERN stack application.

01 — OVERVIEW
Project Details
Why Trendify?
  • Full-stack MERN (MongoDB, Express, React, Node) e-commerce platform
  • Targets Gen-Z users with a sleek dark-mode glassmorphism aesthetic
  • AI-powered fashion assistant using Google's Gemini 2.0 Flash API
  • Voice-controlled navigation using Web Speech API
  • ML-based size recommender (BMI decision tree algorithm)
  • Secure payment gateway via Razorpay + Cash on Delivery support
  • Admin panel with product management, order tracking and analytics
  • Image CDN via Cloudinary for fast, scalable product images

04 — SHOPPING
Product Browsing & Cart
Shopping Flow
📋
Browse Collections
GET /api/product/list fetches all products from MongoDB. Filtered client-side by category, subCategory and search text using React state.
☁️
Cloudinary Images
Product images are served from Cloudinary CDN using URLs stored in MongoDB — supporting up to 4 images per product with a gallery view.
🧠
AI Size Recommender
User inputs height & weight via sliders. A BMI-based decision tree algorithm instantly calculates the ideal size (S/M/L/XL) with a 94% confidence score shown.
🛒
Add to Cart
POST /api/cart/add saves productId + chosen size to the user's Cart Collection in MongoDB. Cart persists across sessions.

AI Features

AI-Powered
Experience

Trendify integrates two AI systems — a Generative AI Fashion Stylist powered by Google Gemini and a Voice Navigation Assistant using the Web Speech API.

06 — AI
Artificial Intelligence
AI Integrations
✨
AI Fashion Stylist
Floating chat widget powered by Gemini 2.0 Flash. Users ask fashion questions and receive personalized outfit advice in real-time. Includes a smart rule-based fallback for rate limits.
🎙️
Voice Assistant
Web Speech API captures voice commands. Intent detection navigates users to pages (Home, Cart, Collections, Contact) or performs in-app searches — entirely hands-free.
🧠
Size Recommender ML
A BMI-based decision tree algorithm takes height (cm) & weight (kg) inputs and recommends S/M/L/XL. Shows a confidence percentage and auto-selects the size.

06 — ADMIN
Administration
Admin Panel Features
🔐
Secure Admin Login
Separate admin credentials in .env. Issues a scoped Admin JWT token with elevated permissions.
📦
Product Management
Add, edit and delete products. Images uploaded via Multer → Cloudinary. Supports 4 images per product.
📊
Analytics Dashboard

Order Management

Job Applications

View registered users, their cart data and complete order history from the admin interface.

Platform Highlights
